Tank Game (Java)

A friend and I built Tank Game in Java using the Swing framework.

We built as practice in writing object-oriented code, designing classes in a way that kept them organized, encapsulated, and reusable. This allowed us to code efficiently and reuse code in other games.

We wanted Tank Game to be a fun game, in addition to being good practice in programming. We made the UI simple and straightforward so games can be played quickly, and the game can be restarted quickly after playing to make it easier to play. We also added destructible walls, weapons, and powerups to add some diversity to the game.

Tank Game is a 2-player tank shooter, where two people can play against each other via local multiplayer. Destroy your opponent's tank by damaging it with weapons, and deplete all of your opponent's lives to win the game. Collect weapons and powerups to help you win!

Galactic Mail (Java)

A friend and I built Galactic Mail in Java using the Swing framework.

Galactic Mail was made using some basic elements from Tank Game, allowing us to reuse code for object movement, collision detection, vector logic, game clock, screen-drawing logic, and controls.

Like Tank Game, we wanted Galactic Mail to be fun as well. Galactic Mail continues endlessly, so you can't "win" the game. So we implemented a high score screen, like those used in old arcade games, which makes the goal of the game to beat your last high score.

Galactic Mail is a 1-player game. Earn points by launching your rocket and landing on other moons. Your rocket is safe while docked on a moon, but when you launch, you'll have less control of your rocket and you'll have to avoid asteroids. Land on all the moons to continue to the next level. Try to beat the high score!
